# OOAD-campus-booking-system# Campus Booking System

## Description
This project is a web-based system for booking campus resources such as rooms, labs, and equipment.

## Features
- User management (Student, Faculty)
- Resource management (Room, Lab, Equipment)
- Booking system
- Approval/Reject functionality
- Status tracking (PENDING, APPROVED, REJECTED)
- REST API using Spring Boot
- H2 Database

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- HTML, CSS, JavaScript

## How to Run
1. Run `BookingApplication.java`
2. Open: http://localhost:8080
